  2. 考虑状态变化

    dp[i][j] 有3种变化

    • dp[i-1][j-1] : word1[i] 修改为 word2[j]，注意当word1[i]==word2[j]时，编辑距离不变，否则编辑距离+1
    • dp[i-1][j] : word1 插入字符 或 word2[j] 删除字符，编辑距离+1
    • dp[i][j-1] : word1 删除字符 或 word2[j] 插入字符，编辑距离+1
